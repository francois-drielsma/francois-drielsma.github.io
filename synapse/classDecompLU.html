Detailed Description

template<typename T>
class DecompLU< T >

Defines the LU decomposition algorithm of a square matrix.

The matrices L and U are such that or with

  • is the permutation applied to the matrix to decompose;
  • is a lower triangular matrix of diagonal elements unity;
  • is an upper triangular matrix.
