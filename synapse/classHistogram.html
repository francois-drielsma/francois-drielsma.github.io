Detailed Description

Data stucture for histograms of arbitray large dimensions.

Produces a bucketing, provided with a set of data points in n dimensions. Returns the binning, their contents and densities.

The underlying structure is a Grid object. A histogram of bins is simply a grid of points that separates the right amount of cells.
