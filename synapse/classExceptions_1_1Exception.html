Detailed Description

Global exception object for all non-standard exceptions thrown by the code.

Exception has a severity (exceptionLevel), an error message and a location. Also a function to return the stack trace (gcc only).

If not using gcc, define NO_STACKTRACE at compile time

Note that the throw() directives are there to explicitly declare that exceptions cannot be thrown in the constructors and destructors
