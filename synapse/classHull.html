Detailed Description

Data stucture of a convex hull.

The object orgQhull::Qhull cannot be copied and hence makes it hard to memory handle. This class stores the basic parameters computed by Qhull to be restituted on command. 

Qhull performs the heavy lifting, this is a data stucture class.
