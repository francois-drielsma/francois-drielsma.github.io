Detailed Description

Computes the Delaunay triangulation of a set of points.

The Delaunay triangulation is obtained by computing the (n+1)-dimensional convex hull of the points by adding the L2 vector norm as an additional coordinate. This forms a paraboloid which projected bottom end corresponds to the triangulation. Qhull performs the heavy lifting, this is a front end class.
