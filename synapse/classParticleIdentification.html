Detailed Description

Performs particle identification.

Currently only one implemented identification method (TOF).

Time-of-flight particle identification:

  1. fills a histogram with the time-of-flights between TOF stations i and j;
  2. fits the muon and pion peaks with a two-peak Gaussian;
  3. determines the time limits between the e, mu and pi peaks.
  4. when called, checks in which interval the time-of-flight falls.

Returns the particle species as a Geant4 particle ID:

  • Electron/positron : 11;
  • Muon/Antimuon: 13;
  • Charged pion: 211.
