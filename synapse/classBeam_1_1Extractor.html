Detailed Description

Extracts the data from an array of imported ROOT file.

Stores the list of data files to be analyzed by phase space evolution algorithms. Upon request, loops over the list the list of files provided and looks for the requested data type.

May be requested to limit the extraction to a set of virtual planes in the simulation or a set of stations in the data.
