Detailed Description

Defines the methods associated with the Spiral distribution in any dimension.

This distribution is based on an n dimensional Gaussian that has a radial dependant rotation applied to it. The rotation preserves the normalisation of the Gaussian. It adopts the shape of a spiral in any 2 dimensional projections. It is a Fermat spiral as the rotation angle grows as .

The radius of this density function is defined as the product, , with the phase space vector, the covariance matrix and the twisting matrix, function of the squared radius, .
